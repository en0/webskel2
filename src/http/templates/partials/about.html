SKEL Flask API/UI with AngularJS/Bootstrap

This is a skeleton for Flask drivin AngularJS with Bootstrap application.

/templates/index.html
The site template for angularjs. Loads angular, bootstrap and site files
/templates/partials/*
All the partials for the angularjs views
/static/js/*
Site specific javascript files
/static/lib/*
All 3rd party libraries.
/static/css/site.css
Site wide CSS files (empty).

Since AngularJS and jinja2 use the same varialbe interpolation, ie: 

{{ "{{ VARIABLE }}" }}
The AngularJS interpolation has been modified to: 
[[ StringConst ]]
If you wish to use a diffrent delimiter, you can modify this character in /static/js/app.js
