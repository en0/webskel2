SKEL Flask API/UI with AngularJS/Bootstrap

This is a skeleton for a Flask drivin AngularJS application with Bootstrap.

/src
All project source code.
/src/libs
Utilites, Models, Wrappers, Helpers, and Carmon Sandiago.
/src/webapp
The codes related to servicing web content.
/src/webapp/api
A Flask blueprint for all things related to the API.
/src/webapp/ui
A Flask Blueprint for all things related to the UI.
/src/webapp/ui/templates/
Main page index where all our client side code is loaded.
/src/webapp/ui/templates/partials/*
All the partials for the angularjs views
/src/webapp/ui/static/js/*
Site specific javascript files
/src/webapp/ui/static/css/site.css
Site wide CSS files (empty).
/src/webapp/static/vendor/*
All 3rd party client libs (bootstrap, angular).

Since AngularJS and jinja2 use the same varialbe interpolation, ie: 

{{ "{{ VARIABLE }}" }}
The AngularJS interpolation has been modified to: 
[[ StringConst ]]
If you wish to use a diffrent delimiter, you can modify this character in /src/webapp/ui/static/js/app.js
